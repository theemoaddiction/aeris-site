AERIS MANDRAKE GEOCITIES SITE

Upload everything in this folder to GitHub Pages, Cloudflare Pages, Netlify, or Neocities.

For GitHub Pages:
1. Create a new repo.
2. Upload all files/folders from this package.
3. Go to Settings > Pages.
4. Source: Deploy from branch > main > /root.
5. Add custom domain: myliminalreality.com.
6. In GoDaddy DNS, point the domain using the records GitHub gives you.

Audio:
- Put public preview files into /audio/.
- Put original Jet-Set Stereo archive tracks into /audio/owns-your-brain/.
- Update the src paths in index.html and owns-your-brain.html.

Main files:
- index.html
- owns-your-brain.html
- catalog.html
- style.css
- script.js
- /assets/ images
- /audio/ placeholders
